# Food Price Analytics Dashboard (Power BI)

This project is an interactive dashboard analyzing food commodity prices in the Philippines.  
It visualizes trends, regional comparisons, and price changes over time to help understand wage and commodity patterns.

## Overview

The dashboard provides:
- Retail and wholesale price comparisons
- Year-over-year trends
- Regional breakdown of prices
- Commodity category filtering
- Interactive visualizations for better analysis

## Tools Used

- Power BI
- Excel (data cleaning and preparation)

## Features

- Trend charts for food commodity prices
- Regional analysis and comparisons
- Category filtering and slicing
- Data cleaning and preparation before visualization
- Dashboard layout designed for clarity and usability
